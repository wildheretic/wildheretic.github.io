You may not have heard of this term before. NPCs stands for Non Player Characters. These are characters in a video game that the computer code generates to make the game more interesting and challenging for the people playing. They make the game what it is.

There is an idea that at least half the world's population are actually not human, but characters solely made up by this simulation / dream / game / soul construct that we live inside. In other words, not everyone has access to an independent soul and/or mind.

In fact, a true NPC would have no access to a soul, only a physical body and perhaps a phantom body. They could be termed "stocking fillers". Dolores Cannon called them "backdrop people".
